# System Configuration Reference for Claude Code

This file documents the system organization and configuration paths for AI assistants.

## XDG Base Directory Compliance

This system follows the [XDG Base Directory Specification](https://specifications.freedesktop.org/basedir-spec/basedir-spec-latest.html).

**IMPORTANT:** Configuration files are located in `~/.config/`, NOT in the home directory root.

## Configuration File Locations

### Shell Configuration
- **Zsh config:** `~/.config/zsh/.zshrc`
- **Zsh variables:** `~/.config/zsh/variables.zsh`
- **Zsh aliases:** `~/.config/zsh/aliases.zsh`
- **Zsh environment:** `~/.zshenv` (in home root, sets XDG paths)
- **Oh-My-Zsh:** `~/.config/zsh/oh-my-zsh/`

### Terminal & Multiplexer
- **Tmux config:** `~/.config/tmux/tmux.conf`
- **Terminal:** Uses tmux in persistent sessions (auto-attach to "main")

### Editor
- **Neovim config:** `~/.config/nvim/`
- **Uses:** LazyVim setup with lazy.nvim plugin manager

### Development Tools
- **Git config:** `~/.config/git/`
- **GitHub CLI:** `~/.config/gh/`
- **FZF:** `~/.config/fzf/`

### macOS Specific
- **Hammerspoon:** `~/.config/hammerspoon/`
- **Yabai:** `~/.config/yabai/`
- **Karabiner:** `~/.config/karabiner/`

## SSH Configuration

- **SSH keys:** Managed by Strongbox password manager
- **SSH agent socket:** `~/.config/strongbox/agent.sock` (symlink to Strongbox's agent)
- **Strongbox agent location:** `~/Library/Group Containers/group.strongbox.mac.mcguill/agent.sock`

## Environment Variables

Key environment variables set in `~/.zshenv`:
- `XDG_CONFIG_HOME=~/.config`
- `XDG_DATA_HOME=~/.local/share`
- `XDG_CACHE_HOME=~/.cache`
- `XDG_STATE_HOME=~/.local/state`
- `ZDOTDIR=~/.config/zsh`

## Dotfile Management

- **System:** chezmoi
- **Source directory:** `~/.local/share/chezmoi/`
- **Remote repo:** `thomaskgb/dotfiles`

## Platform

- **OS:** macOS (Darwin 23.5.0)
- **Shell:** zsh with Powerlevel10k theme
- **Package manager:** Homebrew

## Search Strategy

When looking for configuration files:
1. **Always check** `~/.config/<tool-name>/` first
2. Check `~/.local/share/<tool-name>/` for data files
3. Only check `~/.<config>` as last resort (this system avoids root dotfiles)
4. For chezmoi-managed files, check `~/.local/share/chezmoi/dot_config/`
